---
title: "Create Product"
api: "POST /products"
description: "Create a new product (Admin only)"
auth: "Bearer token with ADMIN role required"
---

## Authentication

This endpoint requires authentication with a JWT token and ADMIN role.

```bash
Authorization: Bearer <token>
```

## Body Parameters

<ParamField body="name" type="string" required>
  Product name (cannot be empty)
</ParamField>

<ParamField body="slug" type="string" required>
  URL-friendly unique identifier (cannot be empty, must be unique)
</ParamField>

<ParamField body="description" type="string" required>
  Product description (cannot be empty)
</ParamField>

<ParamField body="price" type="number" required>
  Product price (must be >= 0, maximum 2 decimal places)
</ParamField>

<ParamField body="categoryId" type="number" required>
  ID of the category (must be >= 1)
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the created product
</ResponseField>

<ResponseField name="name" type="string">
  Product name
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly unique identifier
</ResponseField>

<ResponseField name="description" type="string">
  Product description
</ResponseField>

<ResponseField name="price" type="decimal">
  Product price
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Product active status (defaults to false for new products)
</ResponseField>

<ResponseField name="categoryId" type="number">
  ID of the category this product belongs to
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when the product was created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when the product was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.kiototeteria.com/products \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Oolong Tea",
    "slug": "oolong-tea",
    "description": "Traditional oolong tea with smooth flavor",
    "price": 11.99,
    "categoryId": 1
  }'
```

```javascript JavaScript
const response = await fetch('https://api.kiototeteria.com/products', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Oolong Tea',
    slug: 'oolong-tea',
    description: 'Traditional oolong tea with smooth flavor',
    price: 11.99,
    categoryId: 1
  })
});
const product = await response.json();
```
</RequestExample>
