---
title: "Update Product Status"
api: "PATCH /products/:id/status"
description: "Update the active status of a product (Admin only)"
auth: "Bearer token with ADMIN role required"
---

## Authentication

This endpoint requires authentication with a JWT token and ADMIN role.

```bash
Authorization: Bearer <token>
```

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the product to update
</ParamField>

## Body Parameters

<ParamField body="isActive" type="boolean" required>
  New active status for the product
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the product
</ResponseField>

<ResponseField name="name" type="string">
  Product name
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly unique identifier
</ResponseField>

<ResponseField name="description" type="string">
  Product description
</ResponseField>

<ResponseField name="price" type="decimal">
  Product price
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Updated active status of the product
</ResponseField>

<ResponseField name="categoryId" type="number">
  ID of the category this product belongs to
</ResponseField>

<ResponseField name="createdAt" type="datetime">
  Timestamp when the product was created
</ResponseField>

<ResponseField name="updatedAt" type="datetime">
  Timestamp when the product was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH https://api.kiototeteria.com/products/1/status \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "isActive": true
  }'
```

```javascript JavaScript
const response = await fetch('https://api.kiototeteria.com/products/1/status', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    isActive: true
  })
});
const product = await response.json();
```
</RequestExample>
